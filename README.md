# CV ديناميكي لمجال الـ IT

## الطريقة السريعة
1. أنشئ مستودع جديد على GitHub باسم `cv` (أو أي اسم).
2. أضف الملفات: `index.html`, `data.js`, `styles.css`, `README.md`.
3. حرّر `data.js` وأدخل بياناتك.
4. فعّل GitHub Pages:
   - ادخل إلى Settings -> Pages.
   - اختر Branch: `main` و`/ (root)`.
   - احفظ؛ سيظهر رابط موقعك العام بعد قليل.

## تعديل سريع
- كل بيانات السيرة في `data.js`. عدّل النصوص والروابط فقط.
- لا تحتاج لتعديل `index.html` أو `styles.css` إلا إذا رغبت بتخصيص التصميم.

## نصائح للوظائف
- حدّث الخبرات والمشاريع لتبرز نتائج قابلة للقياس (KPIs، زمن استجابة، تخفيض تكاليف).
- أضف وسوم تقنية (Tags) لكل خبرة لتسهيل القراءة الآلية (ATS).
- اربط مشاريع GitHub مباشرة وروّج لأمثلة عملية.
